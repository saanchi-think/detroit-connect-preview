# Detroit Connect Interactive Preview

A public, database-free product preview of Detroit Connect. It mirrors the
current visual direction and provides interactive navigation, AI-style search,
connector filtering, favorites, friend recommendations, forum posting, and an
Explore Detroit map.

The production application remains in `../detroit-connect-app`; this preview is
intentionally self-contained so it can be shared for client feedback before the
Supabase project is connected.

## Commands

```bash
npm install
npm run dev
npm run build
npm test
```
